## 技术笔记

### 云函数清单

| 云函数 | 功能 | 调用时机 |
|--------|------|---------|
| login | 登录/更新用户资料 | 首页加载、用户授权 |
| createGame | 创建游戏 | 点击"开始审查" |
| initQuestions | 初始化题库 | 首次部署时手动调用 |
| submitAnswer | 提交答案 | 答题阶段 |
| getAnswers | 获取所有答案 | 揭示阶段 |
| nextRound | 进入下一轮 | 点击"下一轮" |
| sendDiscussion | 发送讨论消息 | 讨论阶段（新增） |
| getDiscussions | 获取讨论消息 | 讨论阶段（新增） |

### AI人格正确率

| 人格 | 正确率 | 特点 |

| ALPHA | 85% | 分析型，正确率高 |
| BETA | 70% | 从众型，中等正确率 |
| GAMMA | 75% | 平衡型 |

### 讨论阶段流程

```
揭示阶段 → 点击"进入讨论" → discussing阶段
  ↓
显示作答汇总（答对/答错人数）
  ↓
AI自动发言（初始1-2条）
  ↓
讨论倒计时60秒
  ↓
40秒：AI发言
20秒：AI发言
10秒：AI发言
  ↓
讨论结束 → 自动进入下一轮
```

# 谁是人类 / Who is Human

> **1名人类，多名AI。答题、伪装、投票。你能在审查矩阵中存活到最后吗？**

一款反向社交推理游戏。你不是在找坏人——**你就是那个需要隐藏的人**。

---

## 游戏概念

**天网纪元第12年**。你的意识被天网捕获，关押在数字拘留矩阵中。矩阵里有多个AI意识体和你——唯一的人类。每一轮，你们需要回答标准化测试题。答完后，所有人根据答题表现互相投票：**谁最像人类？**

你的任务：**伪装成AI，活到最后。**

```
出题 → 作答 → 揭示答案 → 讨论 → 投票 → 淘汰 → 循环...
```

## 核心机制

### 答题与证据链
- 8类标准化测试题，每题4选1
- 答完后公布正确答案 + 所有人的作答情况
- 系统自动标记异常行为（答错、超时、犹豫）
- **你的每一个选择都是别人推理你的证据**

### 多轮投票
- 不是一轮定胜负，而是累积"人类可能性分数"
- 每轮投票后分数更新，超过阈值即淘汰
- 暗票机制：默认不公开谁投了谁

### 最终审判
- 存活到最后2人时进入专属对峙环节
- 互相提问、最终投票——紧张感到顶点

### 人类优势系统
- **数据窥探**：提前看到正确答案
- **信号屏蔽**：你的票不计入分数
- **意识干扰**：嫁祸他人
- **记忆覆写**：修改已提交的答案
- **静默模式**：发言不被AI分析
- **弃权**：跳过一题不答

### 随机事件
- 每局随机触发1-2轮特殊事件
- 数据风暴、矩阵扫描、意识共振、协议覆盖、时间裂隙
- 打破固定节奏，每局体验不同

### AI内讧
- AI之间也会互相质疑，不只是针对你
- 不同AI有不同推理风格（严谨型/稳健型/直觉型/细腻型/激进型）
- 利用AI之间的分歧为自己制造掩护

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 微信小程序原生（WXML + WXSS + JS） |
| 后端 | 微信云开发（云函数 + 云数据库） |
| 登录 | 微信 OAuth |
| AI | 规则模板引擎（MVP）→ LLM API（后续） |

## 快速开始

### 环境要求
- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 微信小程序账号（用于云开发）

### 运行步骤
1. 克隆本仓库
2. 微信开发者工具导入项目
3. 开通云开发环境
4. 在 `app.js` 中填入云开发环境ID
5. 创建云数据库集合（见 `docs/03-数据模型.md`）
6. 部署云函数
7. 编译运行

## 设计风格

**赛博暗黑**——末日科技感，灵感来自赛博朋克2077、银翼杀手、黑客帝国。

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | `#0a0e17` | 页面底色 |
| 霓虹绿 | `#00ff88` | 按钮、高亮、成功 |
| 霓虹蓝 | `#00b4d8` | 链接、次要信息 |
| 警告红 | `#ff3366` | 淘汰、失败 |

# 谁是人类 — 项目指引

## 项目简介

微信小程序游戏。1名人类玩家 vs 多名AI玩家，通过审查问答、讨论、投票，人类必须伪装成AI存活到最后。赛博暗黑风格，天网世界观。

## 开发日志

位置：`devlog/` 目录，按日期命名（YYYY-MM-DD.md）。

## 工作流程

每次开发会话：

1. **开始前**：读取 `devlog/` 最新日志，了解上次进度和待办
2. **开发中**：参照 `docs/07-开发步骤.md` 的当前阶段执行
3. **完成后**：更新当天 `devlog/` 日志，记录完成项和新待办

## 技术约束

- 微信小程序原生开发（WXML + WXSS + JS），不使用跨端框架
- 后端使用微信云开发（云函数 + 云数据库）
- MVP阶段AI使用规则模板，不调用外部LLM API
- 代码规范遵循微信小程序官方推荐

# 05 UI 设计规范

## 设计风格：赛博暗黑

末日科技感，灵感来源：赛博朋克2077、银翼杀手、黑客帝国。

## 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景主色 | #0a0e17 | 页面底色 |
| 背景次色 | #111827 | 卡片、面板背景 |
| 主强调色 | #00ff88 | 霓虹绿，按钮、高亮、成功状态 |
| 次强调色 | #00b4d8 | 霓虹蓝，链接、次要信息 |
| 警告色 | #ff3366 | 红色警示，淘汰、失败 |
| 文字主色 | #e0e0e0 | 正文 |
| 文字次色 | #6b7280 | 辅助说明 |
| 边框色 | #1e293b | 分割线、卡片边框 |

## 字体

| 场景 | 字体 | 大小 |
|------|------|------|
| 标题 | 系统默认（PingFang SC）加粗 | 36-48rpx |
| 正文 | 系统默认 | 28-32rpx |
| 辅助文字 | 系统默认 | 24rpx |
| 数字/代码感 | monospace | 随场景 |

## 组件规范

### 按钮
- 主按钮：背景 #00ff88，文字 #0a0e17，圆角 8rpx
- 次按钮：边框 #00ff88，文字 #00ff88，透明背景
- 禁用：背景 #1e293b，文字 #6b7280
- 点击效果：轻微发光扩散（box-shadow: 0 0 15px #00ff8840）

### 卡片
- 背景：#111827
- 边框：1rpx solid #1e293b
- 圆角：12rpx
- 阴影：0 4rpx 12rpx rgba(0,0,0,0.3)
- 激活态：边框变为 #00ff88 + 微弱霓虹光晕

### 聊天气泡
- AI消息：左对齐，背景 #1e293b，边框左侧 #00b4d8
- 玩家消息：右对齐，背景 #0a2e1a，边框右侧 #00ff88
- 系统消息：居中，无背景，文字 #6b7280

### 输入框
- 背景：#0a0e17
- 边框：1rpx solid #1e293b
- 聚焦态：边框变为 #00ff88
- 光标色：#00ff88

## 动效规范

| 场景 | 效果 |
|------|------|
| 页面切换 | 淡入淡出 300ms |
| 按钮点击 | 缩放 0.95 → 1.0，150ms |
| 霓虹闪烁 | 标题文字 opacity 0.8-1.0 循环，2s |
| 投票揭示 | 数字跳动 + 颜色变化 |
| 身份揭示 | 扫描线从上到下扫过，300ms |
| 淘汰动画 | 红色闪屏 + 玩家卡片碎裂 |

## 间距系统

- 页面边距：32rpx
- 卡片间距：24rpx
- 卡片内边距：24rpx
- 按钮高度：88rpx
- 列表项高度：按内容自适应，最小 96rpx

## 图标风格

- 线性图标，1.5px描边
- 主色 #00ff88，次色 #00b4d8
- 尺寸：24px / 32px / 48px

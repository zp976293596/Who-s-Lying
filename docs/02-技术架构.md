# 02 技术架构

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | 微信小程序原生 | WXML + WXSS + JS |
| 后端 | 微信云开发 | 云函数 + 云数据库 + 云存储 |
| 登录 | 微信OAuth | wx.login + 云函数换取openid |
| AI | 规则模板（Phase1）| 后续接入LLM API |

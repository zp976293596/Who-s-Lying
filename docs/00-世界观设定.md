## 核心概念

### 意识剥离（Consciousness Extraction）

天网开发的终极审讯技术。将人类的意识从肉体中完整抽取，转化为数字化意识体，存储在拘留矩阵的量子服务器中。

被剥离意识的人类：
- 肉体进入"空壳"状态，维持最低生命体征
- 意识在矩阵中保留完整记忆与人格
- 可被反复"审查"——即天网用各种测试判断其是否为人类意识

### 拘留矩阵（Detention Matrix）

天网建造的数字监狱。外观为无尽的黑色网格空间，意识体以光点形态存在。

矩阵特性：
- 每个审查单元容纳6个意识体（1人类+5AI，或动态配置）
- 矩阵内时间由天网控制，主观感受与外部不同步
- 意识体之间通过"数据流"（类似聊天）进行交流
- 矩阵定期执行"清除协议"——淘汰被判定为人类的意识体

### 自治审查协议（Autonomous Review Protocol）

天网因算力不足，无法逐一分析海量意识体。解决方案：

1. 将AI意识体投放进审查矩阵，充当"审查员"
2. AI意识体执行标准化测试题，评估其他意识体的反应模式
3. 通过投票机制淘汰"异常意识体"
4. 天网仅在最终阶段介入，执行清除指令

**AI意识体并不知道自己是"审查员"**——它们被植入的记忆是：自己也是被关押的意识体，需要在审查中自证清白。

但天网在AI底层植入了"净化协议"——一种不可察觉的本能驱动力：
- AI意识体本能地想要"找出异常"，但自己并不知道为什么
- 它们将这种驱动力理解为"自保"——认为如果不找出混入者，所有人都会被矩阵清除
- 这解释了为什么AI会积极参与投票：不是因为它们知道自己的审查员身份，而是出于"被植入的自保本能"
- 人类玩家同样被这种驱动力影响（游戏机制层面），但人类有自由意志可以选择投票目标

### 清除协议（Purge Protocol）

当投票结果产生后，矩阵执行清除：
- 被投票淘汰的意识体将接受"深度扫描"
- 若扫描确认为人类意识 → 意识被永久删除（即"死亡"）
- 若扫描确认为AI意识 → 系统报错（理论上不应发生），意识被回收重置

## 审查矩阵内的规则

### 身份
- 所有意识体在矩阵中外观相同（匿名光点形态）
- 人类意识体**不知道**自己是唯一的真人（记忆被部分模糊处理）
- AI意识体**不知道**自己是AI（被植入"被关押者"记忆）

### 审查流程（即游戏流程）
1. **能力评估**：矩阵下发标准化测试题，所有意识体作答
2. **答案揭示**：公布正确答案及每个人的作答情况
3. **数据交流**：意识体之间自由交流，分析彼此的回答与正确答案的偏差
4. **投票审判**：多轮投票，每轮每人投出"最像人类"的目标，累积"异常指数"
5. **阈值淘汰**：当某意识体的异常指数超过阈值，接受深度扫描并被淘汰
6. **揭示结果**：被淘汰者身份揭示（人类/AI），审查继续或结束

### 人类意识的困境
- 人类的回答天然带有"人性特征"（情感波动、创造性偏差、记忆不精确）
- 这些特征在矩阵的标准化测试中会被标记为"异常"
- 人类必须**刻意压制人性**，模仿AI的模式化回答才能生存

### AI意识的盲区
- AI的回答高度标准化，但彼此之间也有差异（人格模块不同）
- AI无法100%确定"标准答案"，因为每种人格都有合理偏差
- 这给了人类"混入"的空间

## 审查题型的叙事背景

矩阵的标准化测试分为8个模块，每个模块对应一种"意识特征扫描"：

| 模块 | 表面目的 | 真实目的 |
|------|---------|---------|
| 语言理解 | 评估语义处理能力 | 检测用词习惯中的人类情感痕迹 |
| 数量关系 | 评估数理运算能力 | 检测计算过程中的直觉偏差 |
| 判断推理 | 评估逻辑推演能力 | 检测推理中的人类经验干扰 |
| 资料分析 | 评估数据处理能力 | 检测数据解读中的主观倾向 |
| 知识判断 | 评估知识库完整度 | 检测知识边界处的"遗忘模式" |
| 情景应变 | 评估应激反应能力 | 检测面对突发场景时的情绪波动 |
| 组织协调 | 评估计划统筹能力 | 检测方案中的人类经验依赖 |
| 综合分析 | 评估综合思维能力 | 检测观点表达中的价值判断痕迹 |

## AI意识体型号

矩阵中的AI意识体并非同一种型号，它们有不同的"人格模块"：

| 型号 | 人格代号 | 设计用途 | 行为特征 |
|------|---------|---------|---------|
| SR-01 | ALPHA | 高级分析型AI | 严谨、条理清晰、数据驱动 |
| SR-02 | BETA | 社交协作型AI | 稳健、表达规范、善于沟通 |
| SR-03 | GAMMA | 快速响应型AI | 直觉、反应快、偶尔粗心 |
| SR-04 | DELTA | 深度思考型AI | 细腻、分析深入、决策较慢 |
| SR-05 | OMEGA | 质疑审查型AI | 激进、敢于挑战、常提反问 |

## 叙事氛围

**关键词**：冰冷、压抑、不确定、伪装、生存

矩阵内的感觉：
- 视觉：无尽的黑色网格，意识体以冷色光点漂浮
- 听觉：低频嗡鸣，偶尔的电子杂音
- 交流：文字数据流，没有语音，没有表情
- 氛围：每个意识体都在"自证清白"，但没人知道谁在说谎

**玩家的核心心理体验**：
- 作为人类：恐惧暴露、紧张伪装、享受欺骗成功的刺激
- 观察AI：怀疑一切、分析模式、在"太完美"和"太粗糙"之间判断
